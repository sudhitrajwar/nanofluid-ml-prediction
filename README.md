# Machine Learning-Based Prediction of Thermal Conductivity Enhancement in Nanofluids

## Overview

This project focuses on predicting the thermal conductivity enhancement of nanofluids using machine learning techniques. Nanofluids are widely used in advanced heat transfer applications, and accurately predicting their thermal properties is crucial for improving performance in engineering systems.

The project implements multiple regression models and compares their performance with traditional theoretical models.

---

## Objective

* Predict thermal conductivity enhancement using ML models
* Compare performance of different regression algorithms
* Evaluate ML models against classical models (e.g., KKL model)
* Identify the most influential features affecting thermal conductivity

## Results

The machine learning models significantly outperform traditional models in predicting thermal conductivity enhancement.

* SVR achieved the highest accuracy
* Ensemble models (RF, XGBoost) showed strong performance
* Classical model (KKL) showed poor predictive capability
